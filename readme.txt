Description:
Plugin add shortcodes which you can use to display your cryptocurrency wallet address with qr code. 
You can also add donation buttons under all posts.

Use:
1. Instal plugin.
2. Activate plugin
3. Add you wallets and check enabled.
4. Copy shortcode to your post or widget.
5. Check donate all post and type the title.

Important!!!
All default cryptocurrency addresses are our wallets addresses. Change it on your.
There are also links to create your own safe wallet of those cryptocurrencies.

== Frequently Asked Questions ==

= How many shortcodes i can put on page? =
Only one shortcode for one page

= Can I use plugin when I don't have one cryptocurrency wallet? =
Yes, you can enable and disable crypto to display. Also you can create for free new wallet from links under each cryptocurrency section.

= Can I use shortcode on widgets? =
Yes you can.

= What z-index have showing donation popup? =
Z-index of it is 99
